
NGO_FOUNDATION
==============

NGO_FOUNDATION is a dynamic web application developed using ASP.NET MVC with SQL Server as its database. 
This platform is designed to centralize and streamline funding for multiple NGOs, enabling transparency 
and efficient management of donations. The project includes an admin panel with distinct privileges to manage the platform effectively.

---------------------------------------------------------------
Features
---------------------------------------------------------------

User-Facing Features:
1. Centralized Donation Portal
   - Users can donate to multiple NGOs from a single platform.
   - Secure and easy-to-use interface for donors.

2. NGO Profiles
   - Detailed information about each NGO, including mission, projects, and funding needs.

3. Donation Tracking
   - Users can track their donations and view the impact of their contributions.

Admin Panel Features:
1. Role-Based Privileges
   - Different levels of access for super admin, NGO admins, and support staff.

2. NGO Management
   - Approve, suspend, or update NGO accounts.

3. Dynamic Content Management
   - Manage home page content, featured NGOs, and donation campaigns.

4. Analytics and Reports
   - View funding statistics, donor activity, and NGO performance.

5. Audit Trails
   - Log and monitor all administrative actions for transparency.

---------------------------------------------------------------
Technologies Used
---------------------------------------------------------------

- Backend: ASP.NET MVC
- Frontend: HTML5, CSS3, JavaScript, Bootstrap
- Database: SQL Server
- Authentication: ASP.NET Identity
- Deployment: IIS / Cloud Hosting

---------------------------------------------------------------
Installation Instructions
---------------------------------------------------------------

Prerequisites:
- Visual Studio 2019 or later
- SQL Server Management Studio (SSMS)
- .NET Framework 4.7.2 or higher

Steps:
1. Clone the repository:
   git clone https://github.com/Irhamraza123/NGO_FOUNDATION.git

2. Open the project in Visual Studio.

3. Update the web.config file with your SQL Server connection string.

4. Run the database migrations to set up the required tables.

5. Build and run the application:
   - Use the Visual Studio debug tool to launch the project.
   - The application will open in your default browser.
